# 🛒 Marketplace Recanto Heroico

Projeto de marketplace fullstack rodando em um servidor local (Acer Predator) acessível mundialmente via **Cloudflare Tunnels**.

## 🛠️ Tecnologias
- **Frontend:** HTML5, CSS3 (ML Style), JavaScript Vanilla.
- **Backend:** Node.js, Express, JWT, CORS.
- **Banco de Dados:** PostgreSQL.
- **Deploy:** Cloudflare Tunnel.

## 🚀 Como rodar
1. Configure o banco Postgres.
2. No `/backend`, rode `npm install` e `node server.js`.
3. No `/frontend`, sirva o `index.html`.
4. Configure o túnel com `cloudflared`.
